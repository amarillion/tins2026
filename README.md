# Playing the game

The easiest way is to [Play online](https://amarillion.github.io/tins2026/)

See down below on how to run directly form source.

## How to play

Please be patient with the GUI! Given the time constraints, the GUI is not as user-friendly as it could be. 

The goal of each level is to configure the laser to shoot all the triggies in one swoop.

Place different components and link them using connectors, until the laser moves in the desired path. Then Press the *Fire Laser* button to initiate a laser cycle. If you fail to shoot all the triggies in one cycle, they return to their position.

Place various components, then use *Connectors* to link component outputs to inputs. Ultimately, you want to make a path from the *pulse generator* to the *xy laser coordinate* output.

The *monitor* is a useful debugging tool. Place the monitor and connect it to an output, to see the actual value of that output.

## Running directly from source

Alternatively, you can compile and running from source. You need a recent version of npm installed. Then run:

```
git clone https://github.com/amarillion/tins2026
cd tins2026
npm install
num run dev
```

Then point your browser to http://localhost:5173/ to play the game.
